## 📏 Global Rules

| Rule              | Guideline                                                                 |
| ----------------- | ------------------------------------------------------------------------- |
| People            | ❌ None. Only use your own community photos for that.                     |
| Religious imagery | ❌ No mosques, Quran, Islamic art from stock. Keep what you already have. |
| Max per section   | 1 image                                                                   |
| Aspect ratio      | Consistent: 16:9 for heroes, 3:2 for cards                                |
| File size         | Under 200KB each (use [squoosh.app](https://squoosh.app))                 |
| Format            | `.jpg` for photos, `.webp` for modern optimization                        |
| Alt text          | Descriptive of the object/scene, tied to GAMEC context                    |
| Loading           | `loading="lazy"` on everything below the fold                             |
